# copybook-governance-grid

A focused crate that owns the governance mapping between COBOL support-matrix entries and
runtime feature flags.

## Public API

- `GovernedFeatureBinding`
- `GovernanceSummary`
- `governance_bindings`
- `feature_flags_for_support_id`
- `summarize_governance`
- `all governance rows are explicit and total coverage is asserted by tests`
