# Tutorials

This directory holds **learning-oriented** guides.

Use these when you need to understand how to use copybook-rs for the first time or learn a feature end-to-end.

## Available Tutorials

- **[Getting Started](getting-started.md)**  
  Learn the minimal path to parse a copybook, decode records, and process output.
- **[Enterprise Deployment](enterprise-deployment.md)**  
  Walkthrough for production deployment planning, rollout, and operations checks.

## Learning Paths

### New Users

1. Read [Getting Started](getting-started.md)
2. Run the full sample from the tutorial
3. Validate results with the verification steps in the tutorial
4. Continue with the next relevant how-to:
   - [Error Handling in Production](../how-to/error-handling-production.md)
   - [Configure Security Scanning](../how-to/configure-security-scanning.md)

### Operations Path

1. Read [Enterprise Deployment](enterprise-deployment.md)
2. Review [Production readiness checks](../PRODUCTION_READINESS.md)
3. Review [Troubleshooting Matrix](../TROUBLESHOOTING_MATRIX.md)
4. Review release and governance runbooks:
   - [Release Runbook](../RELEASE_RUNBOOK.md)
   - [Release Process](../RELEASE_PROCESS.md)
